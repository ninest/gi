# gi

> Easily create a .gitignore

## Usage

```bash
# Install it
$ pip install gi

# Add a language framework
$ gi add python

# Or add multiple
$ git add node python

# Clear your gitignore
$ git clear

# Uninstall it
$ pip uninstall pip
```

**Note**: Mac or Linux users may have to use `pip3` instead of `pip`.
